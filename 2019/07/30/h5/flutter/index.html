Mac安装

  1. 下载Flutter SDK

    • Flutter官网下载最新可用安装包
    • 解压到指定的文件目录下~/Documents/MyLab

  2. 配置环境变量
    export PUB_HOSTED_URL=https://pub.flutter-io.cn
    export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn
    export PATH=~/Documents/MyLab/flutter/bin:$PATH

  3. 运行flutter doctor查看是否需要其它依赖来完成安装

  4. 注意事项

    • oh-my-zsh下命令执行会提示 sudo: flutter: command not found 、 zsh: permission denied: flutter
    • bash下命令正常执行
    • Android toolchain、Android Studio均可按命令正常配置
    • Xcode、iOS tools命令行配置会产生中断
    • Command Line Tools 安装包下载地址
