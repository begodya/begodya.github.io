1、Print: Entry, “:CFBundleIdentifier”, Does Not Exist

解决方案 后重启电脑

2、Android 第一次在真机或模拟器中运行，需要先挂 VPN 进行依赖包下载

3、Unable to load script from assets ‘index.android.bundle’ 

第一步（创建缺失的目录）：mkdir android\app\src\main\asset    
第二步（在创建好的目录添加程序需要的文件）：react-native bundle --platform android --dev false --entry-file index.js --bundle-output android/app/src/main/assets/index.android.bundle --assets-dest android/app/src/main/res    
第三步：react-native run-android

4、Text设置圆角背景在iOS下无法显示

ios中的Text组件不支持borderRadius，可在Text组件外层套加View组件进行设置

5、工程迁移导致error: PCH was compiled with module cache path '/Users/BeiWang/Documents/MyLab/ReactNative/ShanZhen/ios/build/ModuleCache.noindex/147FY0L6K85ZP', but the path is currently '/Users/BeiWang/Documents/MyLab/Github/ReactNativeGo/ShanZhen/ios/build/ModuleCache.noindex/147FY0L6K85ZP'

找到iOS工程下的Build目录，删除ModuleCache.noindex，并执行react-native run-ios

6、undefined is not an object (evaluating RNGestureHandlerModule.State’)

npm install –save react-native-gesture-handler

7、动态显示/隐藏底部导航栏

保证TabBar与下层页面平级即可
